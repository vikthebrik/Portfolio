## Overview

A design language that treats monospace type and box-drawing glyphs as first-class
layout primitives. Every color is a token, focus states are always visible, and the
component set is tuned for keyboard navigation first.

It anchors the `design` cluster and links to the fairness audit, whose dashboards
were the first real consumer of the token scale and the accessible data-viz
patterns.
