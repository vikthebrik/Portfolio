## Overview

A visual-inertial SLAM backend built on a sliding-window factor graph. The
contribution is a marginalization scheme that keeps the optimization problem
bounded in size while preserving consistency, giving predictable drift over long
trajectories in GPS-denied environments.

This work seeds the `research` cluster and links directly to the drone (which
runs a lighter variant onboard) and to the real-time CV pipeline that feeds it
features.
