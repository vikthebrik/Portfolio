## Overview

An embeddable graph store with a cost-based planner for multi-hop traversals. The
storage layout is cache-aware so neighbourhood scans stay close to memory
bandwidth, and the planner reorders joins using cardinality estimates.

It shares `performance` with the CV pipeline and `optimization` with the SLAM
thesis, knitting the `tech` and `research` clusters together through tags rather
than explicit links.
